
# RCM Web Server

The Express Backend Server For RCM Mobile Web Application.


## Installation

Install the project in your machine

```bash
  git clone https://github.com/theSwayamDev/rcm-web-server.git
  cd rcm-web-server
  npm install
```

## Starting Server
Start the express server

```bash
    npm run build
```
For Developer Mode
```bash
    npm run dev
```
To Run Directly
```bash
    npm start
```
